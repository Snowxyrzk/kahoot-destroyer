O **Kahoot Destroyer** é uma ferramenta poderosa para acertar todas as perguntas do Kahoot automaticamente, garantindo a maior pontuação possível sem esforço!  

## 🏆 Funcionalidades  
<details>  
  <summary>Mostrar funcionalidades</summary>  
  
  - Descobrir todas as respostas do Kahoot em tempo real  
  - Interface simples e intuitiva  
  - Nenhuma instalação necessária, tudo via navegador  
  - Respostas rápidas para sempre ficar em primeiro lugar  
  - Suporte a múltiplos jogos ao mesmo tempo  
  - Compatível com todas as versões do Kahoot  

</details>  

## 🚀 Como Usar  
<details>  
  <summary>Mostrar passos</summary>  

  ### 🖥️ Passo a passo

  2. **Digite o código PIN** do jogo
